# NLP Homework Tasks

Набор учебных скриптов по NLP на русском языке.

## Файлы и задачи

### `homework_nlp1.py`
Домашняя мини-задача (5-10 минут):

1. **Zipf + длинный хвост**:
   - загрузка русского романа (`.txt`, Project Gutenberg),
   - подсчет `tokens/types`,
   - лог-лог график закона Zipf,
   - оценка длинного хвоста (`f <= 3`).
2. **Сравнение предобработки**:
   - raw,
   - lowercase,
   - замена `ё -> е`,
   - фильтрация коротких слов,
   - лемматизация (`pymorphy3`, если доступен).
3. **Интенты**:
   - базовые интенты (`greeting`, `goodbye`, `ask_weather`, `ask_time`),
   - 2 новых интента: `ask_recommendation`, `express_gratitude`,
   - проверка через `spaCy Matcher` и через `regex`.

### `homework_task3.py`
Мини-задача для семинара (русский):

1. Взять фрагмент русского романа (примерно 30-100k символов).
2. Сравнить частоты:
   - по токенам,
   - по стеммам,
   - по леммам.
3. Сравнить размер словаря и долю длинного хвоста (`<= 3`) для всех представлений.
4. Выбрать 5 предложений и сравнить:
   - POS/DEP/NER от `spaCy`,
   - леммы от `pymorphy3`.
5. Собрать список персонажей (NER `PER`) и их частоты.
6. Найти как можно больше предложений с приветствиями (`regex` + `spaCy Matcher`).

### `homework_nlp2.py`
Большая домашняя работа из двух частей.

#### Часть A - Domain shift / распределения

1. Выбрать русский датасет и 2 домена (в коде: `rus_news_classifier`, темы `Политика` и `Спорт`).
2. Посчитать `top-30` log-odds токенов в обе стороны.
3. Построить графики распределений:
   - длина документов,
   - Zipf (топ-100 токенов).
4. Добавить лемматизацию, стоп-слова, сравнить важные токены.
5. Зафиксировать наблюдения по изменениям.

#### Часть B - Наивный Байес + оценка

1. Выбрать 2 авторов (автоматически берутся самые частые в датасете `IlyaGusev/versa`).
2. Нарезать тексты на чанки (в коде `150` токенов).
3. Обучить `MultinomialNB` и оценить:
   - `accuracy`,
   - `classification_report` (precision/recall/F1),
   - `confusion matrix`.
4. Показать `top-20` важных токенов по `Delta(w)`.
5. Добавить предобработку (леммы, стоп-слова, n-граммы) и сравнить качество.
6. Проверить domain shift (обычный текст -> `stressed_text`) и описать, что ухудшилось и почему.

## Быстрый запуск

1. Установить зависимости (пример):

```bash
pip install requests numpy pandas matplotlib scikit-learn datasets spacy nltk pymorphy3 pymorphy2
```

2. Запускать скрипты по очереди:

```bash
python homework_nlp1.py
python homework_task3.py
python homework_nlp2.py
```

## Примечания

- Некоторые шаги требуют интернет для загрузки текстов/датасетов.
- Если `pymorphy2/pymorphy3` не установлен, лемматизация в скриптах частично пропускается.
- Для воспроизводимости в `homework_nlp2.py` используется `RANDOM_STATE = 42`.
